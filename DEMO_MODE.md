## 🚀 快速开始

### 方式1：浏览UI
1. 打开 http://localhost:3001
2. 点击"开始使用"或"立即注册"
3. 在演示模式下，任意登录信息都可以进入

### 方式2：直接测试Dashboard
1. 打开 http://localhost:3001/dashboard
2. 自动使用演示账号登录
3. 查看预设的任务和目标

## ⚙️ 切换到生产模式

当你准备好使用真实的Firebase时：

1. **配置Firebase项目**
   - 参考 [FIREBASE_SETUP.md](./FIREBASE_SETUP.md)

2. **更新环境变量**
   ```bash
   # 在 .env.local 文件中修改：
   NEXT_PUBLIC_DEMO_MODE=false
   ```

3. **重启服务器**
   ```bash
   npm run dev
   ```

## 📝 演示模式限制

- 数据仅在内存中，刷新后重置
- 多个标签页之间数据不同步
- 不支持真实的用户认证
- 仅供UI测试和功能演示
